# Referencias de Artistas por Estilo Musical do Brasil

Uso do agente: este documento serve como base de referencia para criar conteudos, comparacoes, campanhas, legendas, roteiros e analises por genero musical. Nao trate como ranking definitivo. Use como mapa de artistas influentes, populares ou historicamente importantes.

## Regra de uso

- Nao diga que um artista e "o melhor" de forma absoluta.
- Use termos como referencia, influencia, nome importante, artista popular, artista historico ou artista atual.
- Adapte a resposta por regiao, publico e objetivo do artista cadastrado.
- Para artistas novos, use referencias como inspiracao de posicionamento, nao como copia.

## Como o agente deve usar esta base

Ao receber um artista novo:

1. Identificar genero principal e subgeneros.
2. Selecionar 3 a 5 referencias coerentes.
3. Comparar tom, publico e objetivo.
4. Criar posicionamento original.
5. Gerar ideias de conteudo sem copiar os artistas de referencia.

Exemplo:

Se o artista for de piseiro romântico, usar referencias como Joao Gomes, Tarcisio do Acordeon e Zé Vaqueiro para entender linguagem, mas criar um plano proprio de calendario, visual, storytelling e CTA.
